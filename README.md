# memorycard-Gleb
Quiz App — це застосунок для проходження вікторин або тестів.

Користувачеві послідовно показуються запитання з кількома варіантами відповідей. Він обирає відповідь, після чого програма перевіряє, чи вона правильна, та зберігає результат. Наприкінці тесту застосунок підраховує кількість правильних відповідей і показує підсумковий бал або результат проходження.

Такий застосунок зазвичай використовується для навчання, перевірки знань або розваг.
